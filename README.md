# Smart CSV Analyzer

Clean messy CSV files automatically.

## Features

- CSV cleaning
- Duplicate removal
- Missing value handling
- Dashboard generation
- Automatic insights
- PDF reporting
- Plot generation
- CLI workflow

## Install

pip install -r requirements.txt

## Run

python csv_data_analyzer.py --file input/data.csv

## Outputs

output/

cleaned_data.csv

dashboard.html

column_plot.png

report.pdf

## Stack

Python

Pandas

Plotly

Matplotlib

FPDF
